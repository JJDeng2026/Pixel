# 王国守卫 - 塔防

竖版固定塔防网页游戏。英雄在底部左右移动射击，抵御从上往下进攻的敌人，共20波！

## 功能特性

### 核心玩法
- **竖版塔防**：英雄固定在底部，只能左右移动，子弹向上射击
- **手动瞄准**：子弹不会自动追踪，需要玩家移动英雄来瞄准敌人
- **20波敌人**：史莱姆、骷髅兵、恶魔、巨龙，每波敌人数量和种类递增
- **20级英雄**：击杀敌人获取经验升级，每级可选择强化属性
- **5个关卡**：草原、沙漠、森林、城堡、火山，难度递增

### 技能系统
- **火球术**：全屏随机火球坠落，AOE爆炸伤害
- **冰霜剑**：扇形扩散冰剑，穿透攻击
- 技能有冷却时间，可通过升级减少冷却

### 升级选项（9种）
- 生命强化、攻击强化、攻速提升、速度提升
- 弹幕强化（+穿透）、多重射击（+额外子弹）
- 即时治疗、技能威力、技能冷却

### 成长系统
- **永久强化**：用金币永久提升生命、攻击、攻速、速度、技能伤害、金币加成
- **金币系统**：击杀敌人获取金币，用于永久强化
- **存档系统**：LocalStorage 自动保存进度

### 操作支持
- **PC端**：A/D 或方向键左右移动
- **移动端**：在画布上拖拽手指控制英雄移动
- **暂停**：ESC键或点击暂停按钮
- **技能**：点击底部技能按钮释放

## 快速开始

```bash
# 使用Python启动本地服务器
python3 -m http.server 8080

# 然后访问 http://localhost:8080
```

| 操作 | PC | 移动端 |

## 技术特性

- 纯前端 Canvas 引擎，无框架依赖
- 绿幕抠图（Chroma Key）处理 AI 图片透明背景
- 2D 像素风 AI 生成图片，画风统一
- 响应式缩放，适配不同屏幕尺寸
- 触屏和键盘双输入支持

## 数据存储

- 使用浏览器 LocalStorage 存储
- 刷新页面不丢失进度
- 支持 GM 后台重置数据

## 素材来源

所有游戏素材（背景、英雄、敌人、子弹、技能特效）均由 AI 生成，2D像素风格，画风一致。
