# 💀 亡灵骑士·夏侯惇

Vampire Survivors 风格的暗黑三国网页游戏。扮演亡灵骑士夏侯惇，在无尽敌潮中生存下去！

## ✨ 功能特性

### 🎯 核心玩法
- **幸存者风格**：自动攻击 + 走位躲避，击杀敌人获取经验升级
- **6种武器**：灵魂之刃、暗影弹幕、死亡光环、冥火风暴、冥雷链、冰霜新星
- **武器进化**：每种武器可升到6级，效果不断增强
- **5种敌人**：怨灵、骷髅兵、僵尸、恶魔、妖将(BOSS)
- **对局内升级**：击杀敌人获取经验，升级选择属性强化或新武器

### 💰 成长系统
- **永久强化**：用金币永久提升生命、攻击、速度、攻速、护甲、磁铁
- **金币系统**：击杀敌人掉落金币，用于永久强化
- **最高分记录**：自动保存最高分

### 📱 操作支持
- **PC端**：WASD/方向键移动
- **移动端**：虚拟摇杆控制
- **暂停**：ESC键或点击暂停按钮

## 🚀 快速开始

### 本地运行
```bash
# 使用Python启动本地服务器
python3 -m http.server 8080

# 然后访问 http://localhost:8080
```

### 直接运行
将整个文件夹上传到任何网页服务器，用浏览器访问 `index.html`

## 🎮 操作说明

| 操作 | PC | 移动端 |
|------|-----|--------|
| 移动 | WASD / 方向键 | 虚拟摇杆 |
| 暂停 | ESC | 暂停按钮 |
